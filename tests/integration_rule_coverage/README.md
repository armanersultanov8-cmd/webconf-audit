Интеграционные тесты покрытия правил находятся здесь, когда им нужны
фикстуры сразу для нескольких анализаторов и такие фикстуры слишком
широки для одного сервер-специфичного тестового модуля.

Каталог намеренно содержит этот README, чтобы он не выглядел как
незавершенная заглушка, если локально рядом случайно появляется
`__pycache__`.
